## Algorithms

### 패러다임별 알고리즘

알고리즘 패러다임 이란, 알고리즘이 주어진 문제를 해결하기 위해 채택한 기초가 되는 일반적인 방법 혹은 접근법입니다. 알고리즘이 해결하는 문제나 알고리즘의 동작방식이 완전 다르더라도, 알고리즘의 동작 원칙이 같으면 같은 패러다임을 사용했다고 말할 수 있으며, 주로 알고리즘을 구분하는 기준으로 쓰인다.

- **브루트 포스(Brute Force)** - 가능한 모든 경우를 탐색한 뒤 최적을 찾아내는 방식
